# 🚌 Smart School Bus Tracking & Parent Notification System

A full-stack MERN application that enables real-time school bus tracking, automated attendance management, and instant parent notifications. The system enhances student safety through live GPS monitoring, digital attendance tracking, and seamless communication between schools, drivers, and parents.

## 🌐 Live Demo

🔗 https://school-bustracker.vercel.app/login

## 🚀 Features

* 📍 Real-time GPS bus tracking
* 🗺️ Google Maps integration
* 🔄 Live location updates using Socket.IO
* 👨‍🎓 Student attendance management
* 🔔 Instant boarding and drop-off notifications
* 🚌 Driver and route management
* 📊 Admin dashboard and attendance reports
* 🔐 JWT-based authentication and role-based access control

## 🛠 Tech Stack

| Layer          | Technologies                         |
| -------------- | ------------------------------------ |
| Frontend       | React.js, Vite, Tailwind CSS         |
| Backend        | Node.js, Express.js, Socket.IO       |
| Database       | MongoDB, Mongoose                    |
| Maps           | Google Maps API                      |
| Notifications  | Firebase Cloud Messaging, Nodemailer |
| Authentication | JWT                                  |
